## 分类逻辑：按能力分，不按厂商分

多模态产品的分类有很多种砍法——按厂商（OpenAI/Google/字节）、按闭源开源、按免费付费。但这些分类对你没有直接意义——你关心的不是"谁家的"，而是"能不能解决我的问题"。

本文用 **"你能用它们做什么"** 作为分类标准，把 2026 年的多模态产品分成 6 大品类，共覆盖 **40 款产品**。

## 品类一：全能对话

**在一个聊天界面里，文字+图片+语音+生图全都能干。** 这是用户量最大、竞争最激烈的品类，也是大多数人接触多模态 AI 的入口。

### 国际三巨头

| 产品 | 公司 | 核心多模态能力 | 一句话 |
|------|------|-------------|--------|
| **ChatGPT（GPT-4o）** | OpenAI | 聊天+看图+语音对话+DALL-E 生图+代码解释 | 月活 200 亿，最全能的多模态助手 |
| **Gemini** | Google | 文字+图片+视频理解+语音，原生多模态 | 视频理解最强，Google 全家桶深度整合 |
| **Claude** | Anthropic | 文字+图片理解+PDF 解析+Computer Use | 文档和 UI 理解最强，企业级安全性 |

三者各有侧重。ChatGPT 是最均衡的全能选手；Gemini 的视频理解和长上下文在百万 token 级独占优势；Claude 的文档理解准确度和安全性是企业用户的首选。

### 国内梯队

国内全能对话产品 2026 年竞争白热化，形成了四个梯队：

| 产品 | 公司 | 核心多模态能力 | 一句话 |
|------|------|-------------|--------|
| **豆包** | 字节跳动 | 聊天+看图+语音+生图+生视频（跳转即梦）| 国内月活 3.15 亿第一，中文体验最佳 |
| **通义千问** | 阿里巴巴 | 聊天+看图+语音+文档解析+生图（通义万相）| 阿里生态整合，通义听悟语音转写是独有优势 |
| **Kimi** | 月之暗面 | 聊天+长文档理解（百万字级）+图片理解 | 长文档处理国内最强，学术/研究场景首选 |
| **DeepSeek** | 深度求索 | 聊天+图片理解，开源模型 | 开源路线标杆，V4 版本多模态能力大幅提升 |
| **文心一言** | 百度 | 聊天+看图+生图（文心一格）+语音 | 百度生态集成，搜索+地图+文库加持 |
| **腾讯元宝** | 腾讯 | 聊天+看图+语音+微信生态 | 微信生态独有的语料和场景优势 |
| **通义千问 VL** | 阿里巴巴 | 专注视觉理解的旗舰版本 | OCR 和图表理解能力国内第一 |

此外，**Kimi K2.6**（月之暗面）和 **MiniMax M3**（稀宇科技）作为新一代模型在开源榜和评测中表现亮眼，且都具备多模态能力。**阶跃星辰**的 Step-2 系列也在视觉理解上持续发力。

### 怎么选

- 日常使用：豆包（中文最优）或 ChatGPT（全球最强）
- 视频相关：Gemini（视频理解）或豆包（跳转即梦生视频）
- 文档和学术：Claude（企业文档）或 Kimi（长文档）
- 开源自部署：DeepSeek

## 品类二：图像生成

**输入文字描述，输出高质量图片。** 2026 年这个品类已经从"能出图"卷到了"图好看""文字准""编辑方便"。

### 国际梯队

| 产品 | 公司 | 核心优势 | 一句话 |
|------|------|---------|--------|
| **Midjourney** | Midjourney Inc | 审美天花板，光影质感独一档 | 图像质量公认最高，设计师首选 |
| **DALL-E** | OpenAI | ChatGPT 内置生图，Prompt 跟随度最佳 | 可以一遍对话一边出图迭代，长文字也能准确渲染 |
| **Stable Diffusion 3.5** | Stability AI | 开源，可本地部署，社区生态最大 | 需要定制化、本地私密运行的场景首选 |
| **Flux** | Black Forest Labs | 开源新贵，文字渲染+手指处理领先 | SD 创作者的进阶选择 |
| **Google Imagen** | Google | Gemini 生态内，精度高 | Google 全家桶集成，Cloud 用户方便 |

### 国内梯队

| 产品 | 公司 | 核心优势 | 一句话 |
|------|------|---------|--------|
| **豆包 Seedream 4.0** | 字节跳动 | 中文 Prompt 理解好，4K 高清 | 国内使用最方便，自媒体/电商场景优势明显 |
| **通义万相** | 阿里巴巴 | 电商场景优化，商品图/模特图 | 电商从业者首选 |
| **文心一格** | 百度 | 文心生态内，使用门槛低 | 百度用户一键接入 |
| **即梦 AI（图片模式）** | 字节跳动 | 和豆包/Seedance 联动 | 字节系创作生态的一环 |

### 怎么选

| 场景 | 推荐 | 理由 |
|------|------|------|
| 追求最高画质 | Midjourney | 无可争议的画面质感第一 |
| 日常生图+对话 | DALL-E（ChatGPT 内）| 聊天和生图不割裂 |
| 本地化/隐私要求高 | Stable Diffusion / Flux | 本地运行，数据不外传 |
| 中文+电商 | 豆包 Seedream / 通义万相 | 中文 Prompt 理解更准 |

## 品类三：视频生成

**输入文字或图片，输出视频片段。** 2026 年这个品类经历了一次洗牌——Sora 退出消费市场，国产方案和国际专业工具成为主力。

### 关键事件：Sora 退场

2026 年 4 月，OpenAI 宣布关停 Sora 消费者服务，API 将于 9 月全面停用。官方理由是高运营成本与低内容可用率——日均成本约 1500 万美元，普通用户的生成视频可用率偏低。Sora 的退出标志着一味堆资源的视频生成路线在 C 端暂时走不通。

### 当前主流产品

| 产品 | 公司 | 核心优势 | 一句话 |
|------|------|---------|--------|
| **Runway Gen-4** | Runway | 视频质量最高，专业创作工具 | 海外创意从业者首选，好莱坞级 |
| **可灵 AI** | 快手 | 运动物理感最好，中文 Prompt 理解佳 | 国产视频生成标杆 |
| **Seedance** | 字节跳动 | 人物一致性和表情细节出色 | 与豆包/即梦联动，一条龙创作 |
| **即梦 AI** | 字节跳动 | 字节系视频创作入口，Seedance 后端 | 普通用户最容易上手的视频生成 |
| **Google Veo** | Google | 与 Gemini 打通，画质高 | 目前仅限部分用户和云客户 |
| **Pika** | Pika Labs | 轻量级，创意特效丰富 | 短视频特效和趣味玩法 |
| **Luma Dream Machine** | Luma AI | 3D 场景渲染优势 | 3D 一致性好的视频生成 |

### 怎么选

| 场景 | 推荐 | 理由 |
|------|------|------|
| 追求最高质量 | Runway Gen-4 | 专业级视频输出 |
| 中文 + 通用视频 | 可灵 + 即梦 | 免费额度足，中文体验好 |
| 人物出镜场景 | Seedance | 人物一致性最高 |
| 短视频特效/趣味 | Pika | 创意玩法多 |

## 品类四：语音交互

**让 AI 能听会说。** 从录音转文字到实时语音对话到语音克隆，应用场景极广。

| 产品 | 公司 | 核心能力 | 一句话 |
|------|------|---------|--------|
| **OpenAI Realtime API** | OpenAI | 实时语音对话，WebSocket 流式 | 开发者集成首选，延迟最低 |
| **ElevenLabs** | ElevenLabs | 语音合成+语音克隆，情感控制精细 | 语音质量天花板，多语言支持 |
| **Whisper** | OpenAI | 开源语音识别，本地可部署 | 最好的开源 STT，支持 99 种语言 |
| **通义听悟** | 阿里巴巴 | 中文语音识别+实时转写+AI 纪要 | 国内会议记录场景首选 |
| **Fish Audio** | Fish Audio | 开源 TTS + 语音克隆 | 中文语音克隆开源方案标杆 |
| **GPT-SoVITS** | 开源社区 | 少样本中文语音克隆 | 几秒音频即可克隆，社区活跃 |

### 怎么选

| 场景 | 推荐 | 理由 |
|------|------|------|
| 实时语音对话（开发）| OpenAI Realtime API | 延迟最低，文档完善 |
| 高精度配音/有声书 | ElevenLabs | 语音质量和情感控制最佳 |
| 中文会议转写 | 通义听悟 | 中文识别最准，自动出纪要 |
| 语音克隆 | ElevenLabs（国际）/ Fish Audio（开源） | — |
| 本地离线识别 | Whisper | 免费、开源、多语言 |

## 品类五：音乐音频

**一句话生成完整歌曲，或一段背景音乐。** 这个品类刚起步但体验已经足够惊人。

| 产品 | 核心能力 | 一句话 |
|------|---------|--------|
| **Suno** | 文字→完整歌曲（作词+编曲+人声演唱） | 现象级产品，一句话出一首完整歌 |
| **Udio** | 文字→音乐，声音质感和人声更自然 | Suno 的主要竞品，偏重音乐品质 |
| **海绵音乐** | 腾讯旗下 AI 音乐生成 | 国产方案，中文语境理解更好 |

## 品类六：开发者 API

**不通过聊天界面，直接在代码里调用多模态能力。**

| 产品 | 公司 | 能力范围 | 使用方式 |
|------|------|---------|---------|
| **OpenAI API** | OpenAI | GPT-4o 视觉+语音+生图 | REST API，生态最完善 |
| **Gemini API** | Google | 视觉+视频+长上下文 | REST API，视频理解领先 |
| **Claude API** | Anthropic | 视觉+文档+Computer Use | REST API，企业文档分析首选 |
| **HuggingFace** | 社区 | 开源多模态模型（LLaVA/Qwen-VL/InternVL/CogVLM） | 本地部署或推理 API |
| **Replicate** | Replicate | 托管第三方开源模型 | 一键部署，按量付费 |

## 如何选：一张决策表

| 你的需求 | 最佳选择 | 备选 |
|----------|---------|------|
| 日常聊天+偶尔看图+偶尔生图 | ChatGPT / 豆包 | Gemini、通义千问 |
| 生成高质量图片 | Midjourney | DALL-E、豆包 Seedream |
| 中文+电商生图 | 通义万相 / 豆包 Seedream | 文心一格 |
| 生成视频 | 可灵（中文）/ Runway（海外） | Seedance、即梦 |
| 中文内容创作全链路 | 豆包+即梦（字节系） | 通义全家桶（阿里系） |
| 语音克隆或配音 | ElevenLabs | Fish Audio（开源） |
| 生成一首歌 | Suno | Udio |
| 中文会议记录 | 通义听悟 | — |
| 在代码里调用视觉能力 | OpenAI Vision API | Gemini Vision API |
| 本地部署多模态模型 | HuggingFace + Ollama | Replicate |

## 总结

- 多模态产品按能力分成 **6 大品类**：全能对话（11 款）、图像生成（8 款）、视频生成（7 款）、语音交互（6 款）、音乐音频（3 款）、开发者 API（5 款），共覆盖 **40 款产品**
- **全能品类三足鼎立**：ChatGPT（200 亿月活）、Gemini（7.5 亿月活）、豆包（3.15 亿月活），加上通义/文心/Kimi/DeepSeek 形成国内四梯队竞争
- **视频生成洗牌**：Sora 退出，Runway 和可灵/Seedance 成为新格局，即梦（字节系）降低普通用户使用门槛
- **图像生成走向分化**：Midjourney 做品质、DALL-E 做集成、Stable Diffusion/Flux 做开源、国产做中文适配
- 下一篇文章拆解全能型产品代表 ChatGPT，看它怎么在一个界面里搞定所有模态
