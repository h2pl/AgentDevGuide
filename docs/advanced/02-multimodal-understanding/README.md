# 02 — 多模态理解：感知能力

有了全景认知和机制基础，本章逐项展开模型的感知能力——怎么让模型"看懂"和"听懂"。

## 学习路径

本章按"视觉→听觉"的顺序，覆盖两种最重要的感知通道。

### 1. 视觉理解

[视觉理解](./01-vision-understanding.md) 是目前最成熟、用得最多的多模态能力。从图像分析 API、视频帧提取、Computer Use 架构到视觉 RAG，帮你把"让模型看图"这件事做扎实。

### 2. 语音与音频

[语音与音频](./02-speech-and-audio.md) 是最自然的交互通道。Realtime API、语音 Agent 循环、中断处理、本地 vs 云端——从 API 接入到工程落地，逐层讲清楚。
