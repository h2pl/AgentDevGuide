# 04 — 多模态推理与工程：综合与交付

前三章分别讲了认知、感知、生成。本章是多模态的收尾：怎么把各模态串起来做综合推理，以及从 demo 到生产需要补哪些工程能力。

## 学习路径

本章两篇文章覆盖"综合→交付"的最后两步。

### 1. 多模态推理

[多模态推理](./01-cross-modal-reasoning.md) 是多模态的高级形态——当模型需要同时处理图像、文字、语音做综合判断时，事情比单模态复杂得多。这篇讲清楚跨模态推理的能力边界、多步推理链策略、多模态记忆和评估方法。

### 2. 工程实践

[多模态工程实践](./02-multimodal-in-production.md) 是从 demo 到生产的最后一课：评估指标、成本模型、可观测性、延迟优化、安全治理，以及一份完整的上线 checklist。
