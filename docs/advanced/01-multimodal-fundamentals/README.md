# 01 — 多模态基础：认知与机制

在深入多模态的具体能力之前，先建立两层基础认知：多模态 AI 的全景版图，以及模型内部的工作机制。

## 学习路径

本章是进阶栏目的起点，两篇文章构成"全景→机制"的递进关系。

### 1. 建立全景认知

[多模态 AI 全景](./01-multimodal-landscape.md) 回答最根本的问题：多模态是什么、为什么需要、2026 年的模型各支持哪些模态。读完你会对多模态的技术版图有清晰的判断，知道哪些模态组合已经成熟、哪些还在早期。

### 2. 理解底层机制

[多模态核心机制](./02-core-mechanisms.md) 进入模型内部，讲清楚一个关键区分——多模态 AI 其实是两条完全不同的技术路线（理解侧用 Transformer，生成侧用扩散模型），然后深入理解侧的三个核心机制：编码、融合、对齐。

不需要你自己训练模型，但理解这些机制能帮你在遇到问题时快速定位原因，在选型时做出更好的判断。
